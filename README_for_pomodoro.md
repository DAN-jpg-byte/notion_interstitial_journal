# ⏱️ Pomodoro & Interstitial Journaling System

このシステムは、**ポモドーロ・テクニック**による集中力の維持と、**インタースティシャル・ジャーナリング**による作業記録を完全自動化するためのツールです。

## 🌟 このプログラムでできること

* **集中と休憩のサイクル管理**: 25分の集中と5分の休憩を自動でカウントします。
* **休憩中の誘惑をブロック**: 休憩時間に入ると自動的に**Wi-Fiを切断**し、ネットサーフィンを防ぎます。
* **Notionへの自動ログ送信**: 集中時間が終わるたびに、実行回数とタイムスタンプをNotionデータベースへ自動で記録します。
* **思考の整理**: ジャーナリング画面で「次にやること」や「今の気持ち」を書き出し、スムーズに次のタスクへ移行できます。

---

## 🛠️ 事前準備（環境構築）

### 1. 必要なソフトのインストール
Pythonがインストールされている環境で、コマンドプロンプト（またはターミナル）を開き、以下のコマンドを実行して必要なライブラリを導入してください。

```bash
pip install requests python-dotenv
```

### 2. Notion APIの設定
1. [Notion My Integrations](https://www.notion.so/my-integrations) で新しいインテグレーションを作成し、トークン（シークレット）を取得します。
2. 使用するデータベースのIDをコピーします（ブラウザでデータベースを開いた際のURLの一部です）。
3. プログラムと同じフォルダに `.env` という名前のファイルを作成し、以下のように記述して保存します。

```text
NOTION_API_TOKEN=あなたのNotionインテグレーションキー
DATABASE_ID=送信先のデータベースID
```

## 🚀 はじめての設定ガイド

Windowsの「タスクスケジューラ」を活用して、管理者権限の確認画面（UAC）を出さずに、ショートカットからスマートに起動する設定を行います。

### ステップ1：タスクスケジューラへの登録
1. Windowsのスタートメニューから**タスクスケジューラ**を検索して開きます。
2. 右側のメニューから「基本タスクの作成」をクリックし、名前を「**インタースティシャルジャーナリング**」にします。
3. トリガーは「手動（または指定しない）」で進め、「操作」タブの「プログラムの開始」で以下を指定します。
   * **プログラム/スクリプト**: `pythonw.exe` （※裏で実行するため。通常の `python.exe` でも可）
   * **引数の追加**: `notion_interstitial_journal.py` のフルパス（例: `C:\Users\user\Documents\notion_interstitial_journal.py`）
   * **開始（オプション）**: スクリプトが保存されているフォルダのパス（例: `C:\Users\user\Documents\`）
4. **重要**: タスク作成後、作成したタスクを右クリックして「プロパティ」を開き、「全般」タブの「**最上位の特権で実行する**」に必ずチェックを入れて「OK」を押してください（Wi-FiのON/OFF制御に必要です）。

### ステップ2：起動用ショートカットの作成
1. デスクトップの何もない場所で **右クリック > 新規作成 > ショートカット** を選択します。
2. 「項目の場所」に以下を入力して「次へ」を押します。

```text
schtasks /run /tn "インタースティシャルジャーナリング"
```

3. ショートカットの名前を「Journal & Pomodoro」など分かりやすいものにして完了します。
4. （任意）作成したショートカットを右クリックして「プロパティ」を開き、「アイコンの変更」から時計やノートのアイコンに変更すると使いやすくなります。

---

## 📖 使い方

1. 作成したデスクトップのショートカットをダブルクリックして起動します。
2. ジャーナリング画面とポモドーロ画面が同時に立ち上がります。
3. 「集中開始」ボタンを押すとタイマーが始動します。
4. 25分経過後、**自動でNotionにログが送信され、Wi-Fiが切断**されます。
5. 5分の休憩が終わるとWi-Fiが復帰し、次のサイクルの準備が整います。

---

## ⚠️ 注意事項

* **Wi-Fi名について**: `pomodoro.py` 内の `WIFI_INTERFACE_NAME = "Wi-Fi 2"` の部分は、ご自身のPCのネットワーク接続名と完全に一致しているか確認してください（コントロールパネルの「ネットワークとインターネット」>「ネットワーク接続」から確認できます）。
* **タスク名について**: ショートカットに記載する名前（`"インタースティシャルジャーナリング"`）は、タスクスケジューラで登録したタスク名と一言一句同じにする必要があります。
